# Doorman

Doormanはサーバ・GUIを有するドア管理システムです。

扉側のハードウェアの実装例については後日別レポジトリで公開する予定です。

## 機能

本ソフトウェアでは、以下の機能が実装されています（一部未実装）

* 部屋の情報の登録
* 部屋の利用登録

## 依存関係

本ソフトウェアは以下のソフトウェアについて依存関係があります。
* Python 3.9 or later with pip
* PostgreSQL

また、使用したpipパッケージはrequirements.txtに掲示があります。
